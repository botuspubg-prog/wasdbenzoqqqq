# Telegram Preview Landing (Vite + React + Tailwind)

## Local dev
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```

## Content edits (no code)
Edit **public/config.json** (title, text, stats, links, accent color, etc.) and replace **public/avatar.jpg**.

## Vercel Deploy
1. Push this repo to GitHub.
2. Import the project in Vercel.
3. Framework: Vite (auto-detected)
4. Build Command: `npm run build`
5. Output Directory: `dist`
